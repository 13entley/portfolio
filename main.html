METHOD

Discovery

Before starting, I fathom a clear and direct way to go about completing the project.

Research

Understanding possible obstacles is important, so I gather data on other similar projects or websites to further understand some niches that may not have been filled.

Design

First step after doing research is to understand how the project will be designed. I work within Sketch to create a clever mockup of the project and start working on intriguing logos or custom graphics.

Sketch

Using Sketch, I then start designing the project entirely, creating a well though out presentation of the project and how I envision it to look when completely finished.

Implementation

After the final design is complete, I then get to work on making the design become a reality, adding special transitions or animations that improve overall user experience.
